# Frontend Templates

Este directorio contendrá los archivos de frontend para visualizaciones interactivas:

- **HTML**: Plantillas para mapas interactivos y dashboards
- **JavaScript**: Lógica de interacción y manipulación del DOM
- **CSS**: Estilos para las visualizaciones

## Estructura Futura

```
frontend/
└── templates/
    ├── index.html          # Página principal
    ├── map_viewer.html     # Visor de mapas interactivo
    ├── styles/            # Archivos CSS
    └── scripts/           # Archivos JavaScript
```

## Tecnologías Previstas

- Leaflet.js o Folium para mapas interactivos
- D3.js para visualizaciones de datos
- Bootstrap o framework CSS similar para estilos
